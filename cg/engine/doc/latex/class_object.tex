\doxysubsection{Detailed Description}
Base class used by all the derived classes. 

This class provides basic functionality such as setting an object\textquotesingle{}s name, enforcing required API methods (e.\+g., {\ttfamily \doxylink{class_object_ad0677142acbe0d973064aa917adf0e4d}{render()}}), and assigning a unique identifier to each object instance. It serves as the base for all derived classes.
