\doxysubsection{Detailed Description}
Class that derives from \doxylink{class_node}{Node}. 

This class is responsible for defining a generic camera. It provides methods for managing viewport dimensions, near and far planes, and updating properties when the window size changes. The class is intended to be a base class for specific camera types that implement the \doxylink{class_camera_adf22782aaef1d6491f5a3144f8518380}{update\+\_\+window\+\_\+size} function.
