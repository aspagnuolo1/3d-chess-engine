\doxysubsection{Detailed Description}
Derived class for managing 2D textures. 

This class represents a 2D texture, which can be loaded from a file and rendered. It inherits from the base {\ttfamily \doxylink{class_texture}{Texture}} class, implementing the necessary methods for handling the loading and rendering of 2D textures.
