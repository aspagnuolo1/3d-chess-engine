\doxysubsection{Detailed Description}
Class that derives from \doxylink{class_camera}{Camera}. 

This class implements a camera with an orthographic projection. It is a specific type of camera suitable for rendering scenes without perspective distortion. It overrides methods to handle rendering and adjusting to changes in the window size.
