\doxysubsection{Detailed Description}
The main graphics engine class implemented as a singleton. 

The \doxylink{class_eng_1_1_base}{Base} class initializes the engine, manages the rendering process, handles user input, and manages the scene graph. It provides utilities for FPS calculation, text rendering, camera management, and scene rendering.
