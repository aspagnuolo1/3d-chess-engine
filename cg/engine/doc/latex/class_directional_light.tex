\doxysubsection{Detailed Description}
Represents a directional light source in the scene. 

\doxylink{class_directional_light}{Directional\+Light} is a subclass of \doxylink{class_light}{Light} that represents a light source with a fixed direction. Unlike point or spotlights, a directional light illuminates objects uniformly, simulating light coming from an infinitely distant source like the sun.
