\doxysubsection{Detailed Description}
Class for representing a 3D mesh. 

The \doxylink{class_mesh}{Mesh} class derives from the \doxylink{class_node}{Node} class and adds functionality for handling 3D mesh data, including vertices, normals, texture coordinates, and faces. It can be rendered with its material properties and transformation matrices.
