\doxysubsection{Detailed Description}
Represents a spotlight in the scene. 

\doxylink{class_spot_light}{Spot\+Light} is a subclass of \doxylink{class_attenuation_light}{Attenuation\+Light} that represents a light source emitting a cone-\/shaped beam of light. It includes parameters for direction, attenuation, and cutoff angle, allowing fine control over the spotlight\textquotesingle{}s behavior.
