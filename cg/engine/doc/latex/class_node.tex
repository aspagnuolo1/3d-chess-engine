\doxysubsection{Detailed Description}
Class that derives from \doxylink{class_object}{Object}. 

The \doxylink{class_node}{Node} class represents a hierarchical structure in a scene graph, allowing for the management of transformations, parent-\/child relationships, and final world coordinate computations. It extends the functionality of the \doxylink{class_object}{Object} class by providing features specific to nodes in a scene graph.
