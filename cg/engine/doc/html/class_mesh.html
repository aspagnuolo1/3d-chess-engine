Detailed Description

Class for representing a 3D mesh.

The Mesh class derives from the Node class and adds functionality for handling 3D mesh data, including vertices, normals, texture coordinates, and faces. It can be rendered with its material properties and transformation matrices.

void ENG_API Mesh::render ( glm::mat4 model_view)

Render the mesh.

This function renders the mesh using OpenGL, applying the material and the associated transformation. It iterates through the faces of the mesh, rendering each vertex along with its normal and texture coordinates.

Parameters
model_viewThe transformation matrix to apply to the mesh.
